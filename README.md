# Personal Portfolio

This repository contains the source code for my personal portfolio website.

## 🌐 Live Website

The site is currently live at: **https://piyush-ojha.vercel.app/**

(You can update the deployment in Vercel or by pushing to `main`.)

## 🛠️ Tech Used

- Vite  
- TypeScript  
- React  
- shadcn-ui  
- Tailwind CSS  

## 📖 How to Use This Repository

You can explore this repository to understand how the portfolio is built using modern frontend tools.

If you'd like to update the list of showcased projects, edit `src/lib/projects.ts`.
Each entry supports fields such as `title`, `description`, `imageUrl`,
`tags`, `size` (large/medium/small) and an optional `link` property.  
When `link` is provided the project card becomes a clickable preview opening
in a new tab, ideal for hosting a live demo or code sandbox.
The homepage grid and other components will automatically pick up any edits.

### Run Locally

```bash
git clone https://github.com/Piyush898-tech/piyush-ojha.git
cd piyush-ojha
npm install
npm run dev
```
